# IDE Shortcuts Cheatsheet

Speed is crucial in interviews. You don't want to struggle with navigating your code.

## VS Code Shortcuts (Mac / Windows)

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| **Navigation** | | |
| Go to File | `Cmd + P` | `Ctrl + P` |
| Go to Symbol | `Cmd + Shift + O` | `Ctrl + Shift + O` |
| Go to Definition | `F12` | `F12` |
| Go Back/Forward | `Ctrl + -` / `Ctrl + Shift + -` | `Alt + Left` / `Alt + Right` |
| **Editing** | | |
| Comment Line | `Cmd + /` | `Ctrl + /` |
| Duplicate Line | `Shift + Option + Down` | `Shift + Alt + Down` |
| Move Line | `Option + Up/Down` | `Alt + Up/Down` |
| Multi-Cursor | `Option + Click` | `Alt + Click` |
| Format Document | `Shift + Option + F` | `Shift + Alt + F` |
| **Refactoring** | | |
| Rename Symbol | `F2` | `F2` |
| Extract Function/Var | `Ctrl + Shift + R` | `Ctrl + Shift + R` |
| **Running** | | |
| Run Test at Cursor | (Custom binding recommended) | |
| Debug | `F5` | `F5` |
| Integrated Terminal | `Ctrl + ` ` | `Ctrl + ` ` |

## Tips for Memorization

1. **Pick 3 a day**: Don't try to learn them all at once.
2. **Force yourself**: Disconnect your mouse for an hour to force keyboard usage.
3. **Print this sheet**: Keep it visible on your desk.
