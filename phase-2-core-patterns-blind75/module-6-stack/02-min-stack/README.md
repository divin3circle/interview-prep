# Min Stack

**Difficulty**: Medium  
**Pattern**: Stack  

## Problem Statement

Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

## Approach

Use two stacks: one for values, one for minimums. Or use single stack storing pairs.

**Time**: O(1) for all operations, **Space**: O(n)
