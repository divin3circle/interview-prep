# Phase 6: Career & Interview Success

## Resume Optimization
- **Action Verbs**: "Designed", "Optimized", "Led", "Built".
- **Metrics**: "Reduced latency by 20%", " Scaled to 1M users".
- **Skills Section**: List Languages (Java, Python), Technologies, and Concepts (REST, AWS, CI/CD).

## Portfolio Study
- Host this repository on GitHub.
- Write a blog post explaining *one* hard problem deeply.
- Show consistent commit history.

## The Interview Day Checklist
- [ ] Good sleep (essential for cognitive function).
- [ ] Water and notebook ready.
- [ ] Test internet and camera.
- [ ] Have stories (STAR) prepared for "Tell me about yourself".
- [ ] **Smile** and be curious!

## Salary Negotiation
- Don't give the first number if possible.
- Focus on total compensation (Equity + Bonus + Base).
- Always be polite but firm. "I'm really excited about the team, but I was hoping to land closer to X based on market rates."
