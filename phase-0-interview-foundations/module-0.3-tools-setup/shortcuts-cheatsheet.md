# IntelliJ IDEA Keyboard Shortcuts Cheatsheet

Master these shortcuts to dramatically improve your coding speed during interviews and practice. This cheatsheet is organized by category for easy reference.

---

## Essential Shortcuts (Top 15)

These are the most critical shortcuts you should memorize first.

| Action | Windows/Linux | macOS | Description |
|--------|--------------|-------|-------------|
| **Run** | Shift + F10 | Ctrl + R | Run current file/configuration |
| **Debug** | Shift + F9 | Ctrl + D | Debug current file/configuration |
| **Format Code** | Ctrl + Alt + L | Cmd + Option + L | Auto-format entire file |
| **Auto-Complete** | Ctrl + Space | Ctrl + Space | Trigger code completion |
| **Quick Fix** | Alt + Enter | Option + Enter | Show intention actions/fixes |
| **Generate Code** | Alt + Insert | Cmd + N | Generate getters, constructors, etc. |
| **Search Everywhere** | Double Shift | Double Shift | Search files, classes, symbols |
| **Find** | Ctrl + F | Cmd + F | Find in current file |
| **Replace** | Ctrl + R | Cmd + R | Find and replace in file |
| **Comment Line** | Ctrl + / | Cmd + / | Toggle line comment |
| **Duplicate Line** | Ctrl + D | Cmd + D | Duplicate current line/selection |
| **Delete Line** | Ctrl + Y | Cmd + Backspace | Delete current line |
| **Move Line Up/Down** | Alt + Shift + ↑/↓ | Option + Shift + ↑/↓ | Move line up or down |
| **Rename** | Shift + F6 | Shift + F6 | Rename variable/method/class |
| **Optimize Imports** | Ctrl + Alt + O | Ctrl + Option + O | Remove unused, organize imports |

## Tips for Learning Shortcuts

1. **Disable Mouse for One Day**: Forces you to use keyboard
2. **Install Key Promoter X Plugin**: Shows shortcuts when you use mouse
3. **Practice One Category at a Time**: Don't try to learn all at once
4. **Create Muscle Memory**: Repeat shortcuts until automatic
5. **Print This Cheatsheet**: Keep it visible while practicing

## Customizing Shortcuts

**File → Settings → Keymap**

### Create Custom Shortcuts

1. Search for action
2. Right-click → Add Keyboard Shortcut
3. Press desired key combination
4. Apply

### Recommended Custom Shortcuts

- **Run with Coverage**: Ctrl + Shift + F10
- **Rerun Tests**: Ctrl + Shift + F10
- **Show Intention Actions**: Alt + Enter (ensure this is set)

## Platform-Specific Notes

### Windows/Linux
- **Ctrl** is the primary modifier
- **Alt** for secondary actions
- **Shift** for variations

### macOS
- **Cmd** replaces Ctrl for most actions
- **Option** replaces Alt
- **Ctrl** is used for some specific actions

### Universal
- **Shift** typically means "reverse" or "opposite"
- **Double Shift** is universal for search
- **F-keys** are consistent across platforms
