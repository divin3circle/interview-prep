# Interview Timeline Diagram

## Visual Representation

This diagram illustrates the standard technical interview process from application to offer.

## Interview Stages

### Stage 1: Application & Resume Screen
**Duration**: 1-2 weeks  
**Key Activities**:
- Submit application and resume
- Automated resume screening
- Recruiter initial review
- Technical keywords and experience matching

**Success Criteria**:
- Relevant experience highlighted
- Clear technical skills section
- Quantifiable achievements
- Clean, professional formatting

---

### Stage 2: Phone Screen
**Duration**: 30-45 minutes  
**Key Activities**:
- Recruiter or engineer conducts initial screening
- Basic technical questions
- Behavioral fit assessment
- Availability and logistics discussion

**Success Criteria**:
- Clear communication
- Basic technical competency demonstrated
- Enthusiasm and interest shown
- Professional demeanor

---

### Stage 3: Technical Interview Round 1
**Duration**: 45-60 minutes  
**Key Activities**:
- One or two coding problems
- Algorithm and data structure questions
- Complexity analysis discussion
- Code quality evaluation

**Success Criteria**:
- Systematic problem-solving approach
- Optimal or near-optimal solutions
- Clean, readable code
- Clear verbal communication

---

### Stage 4: Technical Interview Rounds 2-3
**Duration**: 45-60 minutes each  
**Key Activities**:
- Additional coding problems
- System design (for senior roles)
- Deeper technical discussions
- Team fit evaluation

**Success Criteria**:
- Consistent performance across rounds
- Strong signal in multiple areas
- Collaboration and communication
- Handling of increasing difficulty

---

### Stage 5: Final Decision & Offer
**Duration**: 1-2 weeks  
**Key Activities**:
- Interview panel debrief
- Hiring committee review
- Reference checks
- Offer preparation and negotiation

**Success Criteria**:
- Strong positive signals from all interviewers
- No major concerns or red flags
- Level and compensation determination
- Successful negotiation

## Timeline Overview

```
Application → Phone Screen → Round 1 → Rounds 2-3 → Offer
  (1-2w)        (30-45m)      (45-60m)   (45-60m ea)  (1-2w)
```

## Key Insights

**Total Process Duration**: Typically 3-6 weeks from application to offer

**Critical Points**:
- Each stage filters candidates progressively
- Performance must be consistent across all rounds
- Strong signal in one area cannot compensate for weak signal in another
- Communication matters as much as technical ability

**Preparation Focus**:
- Phone screen: Basic technical knowledge and communication
- Round 1: Core algorithms and data structures
- Rounds 2-3: Advanced problems and system thinking
- Throughout: Professional communication and collaboration
